Compile Script for OMNIROM
==========================

Copy compile.sh to android(OMNI) root dir.

Usage: 
======

```bash
bash compile.sh [-d <device>] [-b <clean|installclean|continue> ] [-t] [-u] [-c]
```


-u : Upload Log File to OMNI Paste  <br>
-p : Use pre-built chromium   <br>
-t : Use timestamp  <br>
-c : Use CCACHE  <br>
